This project aggregates Sri Lankan datasets and plots them in an interactive web view.

- nuuuwan lanka_data_timeseries: https://github.com/nuuuwan/lanka_data_timeseries
  - Site: https://nuuuwan.github.io/lanka_data_search/

Build / deploy

- Local dev: `npm run dev`
- The repo includes a GitHub Actions workflow to export the site (via `next export`) and deploy to GitHub Pages.
